# Ingress 工具箱

> 由 Electron + VUE 开发的 Windows 桌面工具箱 Ingress

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn run dev

# build electron application for production
yarn run build

```
